Dom操作元素

我们在利用Dom操作元素的时候，分为三部：首先，获取元素，其次注册元素，最后执行事件。

获取元素

方法：

在使用以下方法获取元素的时候都需要使用 document.方法 的形式

  • getElementById(),该方法是通过元素的ID名称来获取元素的，返回的是一个对象，括号里面的参数是个字符串，因此需要加**’ ‘**。
  • getElementsByTagName(),通过元素的类名来获取元素，返回的是一个对象数组，参数也是一个字符串，当我们需要获取一类元素的时候，需要使用其类名特有的标识符，如获取一个id元素：
  • getElementsByClassName( ),获取设置了class属性的元素，返回的是一个对象数组。
  • getElementByName( ),获取设置了name属性的元素，返回的是一个对象。
  • querySelector( ),使用css选择器获取一个元素，但是只会获取该css元素的第一个，如获取div元素：
  • querySelectorAll( ),获取某一种元素的所有，返回的是一个对象数组；

执行事件

innerText 和 innerHTML

innerText( ),修改元素中的内容，不会识别html的标签。

innerHTML( ),修改的时候会识别html的标签。

这两种方法只能修改div等普通元素里面的值，不能修改表单里面的值

修改相应元素的属性

element.属性 = ‘ ’；

修改表单属性

获取焦点：onfocus

失去焦点：onblur

修改表单元素属性：可操作属性 type value display checked(checked 属性规定在页面加载时应该被预先选定的 input 元素。) selected(selected 属性规定在页面加载时预先选定该选项。)

修改样式属性：

1.行内样式操作：element.style.属性(相当与在行内操作样式，权重较高)

2.类名样式操作：element.className（相当于内联样式框，应用该语句执行修改之后，会该改变元素本来的类名）

排他思想

概念：有一组元素，想要某一个元素某种样式，需要用到循环的排他思想

具体步骤：

1.所有元素全部清楚样式

2.给当前元素设置样式

3.注意顺序不能颠倒
